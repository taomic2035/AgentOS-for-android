# PRD — 功能清单

> 每行 = 一项可独立验收的功能；可追溯到一个版本。

## A. 入口

| ID | 功能 | 版本 | 描述 |
|---|---|---|---|
| F-A1 | 浮窗呼出 | V0.1 | 拖拽吸附边缘，点击展开 |
| F-A2 | 通知 Tile 呼出 | V0.6 | 系统快捷设置磁贴 |
| F-A3 | 长按 Home 呼出 | V0.6 | 注册为系统 Assistant |
| F-A4 | 自定义手势呼出 | V1.0 | 双击电源 / 三指上滑（依赖 ROM 能力） |
