# 版本路线图

> 详细版本与每版任务拆解的真相源是 `/Users/taomic/.claude/plans/eventual-tumbling-perlis.md` 中的 §5 与 §6。本文档为团队 review 用的精简快照。

| 版本 | 目标 | 关键交付 | 工期 |
|---|---|---|---|
| **V0.1 骨架** | 浮窗呼出 + 硬编码跑通"打开腾讯视频搜三体" | App 壳 + 浮窗 + A11y Service + Skill Runner + 1 个写死 Skill | ~1 周 |
| **V0.2 LLM** | 自然语言 → LLM 规划 → 调用已注册 Skill | LlmClient + IntentRouter + tools 协议 + 设置页 | ~1 周 |
| **V0.3 输入** | 语音 + 文字双输入 | SpeechRecognizer 集成 + 输入面板 | ~4 天 |
| **V0.4 录制** | 用户录新 Skill 并复用 | A11y 事件录制器 + DSL 序列化 + Skill 编辑页 | ~1.5 周 |
| **V0.5 习惯** | 同意图二次出现走 Skill 缓存 | 习惯库 + embedding + 偏好抽取 | ~1 周 |
| **V0.6 场景** | 视频 / 购物 / 外卖 / 打车 / 搜索 内置包 | 5 套官方 Skill + 场景 E2E | ~2 周 |
| **V0.7 守卫** | 不可逆操作前置确认 | RiskGuard + 屏幕变化检测 | ~4 天 |
| **V0.8 本地** | 离线 / 无 key 时降级到本地小模型 | MLC-LLM 或 llama.cpp | ~1.5 周 |
| **V0.9 合规** | 隐私政策 / 权限说明 / 数据加密 / 日志脱敏 | 合规文案 + 数据加密 + 日志脱敏 | ~1 周 |
| **V1.0 RC** | 全场景 E2E + 性能 + 上架包 | 灰度脚本 + 商店素材 + 用户文档 | ~1 周 |

合计 ≈ 12 周（不含商店审核耗时）。

## 每版交付物

每个 V0.x 完成后必须有：

- `release-notes-v0.x.md`
- 关键场景录屏 / GIF
- 新增或变更的文档清单
- 自动化测试报告（`scripts/test-report-v0.x.html`）

## 版本门禁

每版进入下一版的前提：

1. 上版本验收清单全部勾选
2. 单元 + 仪器 + E2E 测试全绿
3. 铲屎官 review 通过
