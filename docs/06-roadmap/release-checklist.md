# 发布检查清单（V1.0 RC）

> 每次准备进入 RC / 商用版本时，从头到尾打勾，缺一项不发。

## 功能完整度

- [ ] 五大场景 E2E 全部通过（视频 / 购物 / 外卖 / 打车 / 搜索）
- [ ] 浮窗在 MIUI / HarmonyOS / EMUI / 原生 Android 至少 3 种 ROM 上正常呼出
- [ ] AccessibilityService 启用引导文字针对国内主流厂商专项写过
- [ ] LLM 设置页：base_url / api_key / model / temperature / max_tokens 全部可配
- [ ] Skill 编辑页：录制 / 编辑 / 删除 / 导出 / 导入 完整闭环
- [ ] 设置页"清空所有数据"可一键执行

## 性能

- [ ] APK 大小 < 30MB（不含本地大模型）
- [ ] 冷启动到浮窗呼出 < 2s
- [ ] Skill 命中场景端到端 < 3s
- [ ] 内存常驻 < 80MB
- [ ] 电量消耗 < 同类前台服务平均水平 + 20%

## 安全 / 隐私

- [ ] 隐私政策文案通过法务 review
- [ ] 用户协议 + 必要权限说明在引导页醒目展示
- [ ] LLM 上行内容默认走"摘要"而非全文，用户可见
- [ ] 本地数据库加密（SQLCipher 或 EncryptedSharedPreferences）
- [ ] API key 加密存储（KeyStore）
- [ ] 日志默认脱敏（手机号 / 地址 / 姓名）
- [ ] RiskGuard 关键词白名单覆盖：付款 / 转账 / 删除 / 发送 / 取消订单 / 退款
- [ ] 不可逆操作前必弹确认（付款页面强制等待用户输入）

## 兼容性

- [ ] minSdk 26 / targetSdk 35 在真机模拟器双跑通过
- [ ] 暗色模式正常
- [ ] 中英双语
- [ ] 横竖屏切换不崩溃
